- **Where** — `R/data/day-03-six-processes.R` (extended — adds `process_B1`/`B2`, `process_C1`/`C2`, `process_D1`/`D2`, `process_E1`/`E2`, `process_F1`/`F2` plus a `six_processes` named list that drives the panel); `R/functions/main-functions.R` (added `mr_limits()` and `six_processes_panel()`); `content/days/day-03/06-six-processes.qmd`; one PNG deleted from `assets/images/day-03/`.
- **Source reference** — Neave Day 3 page 19. Process recipes are described in the page-19 prose: A = total of four dice; B = number of heads when 25 coins are tossed; C = number of red beads in the paddle; D = breakfast-time pulse-rate over 24 consecutive days; E = average measurements on 24 small samples of manufactured cigarette-lighter sockets (Japanese case study); F = monthly United States trade deficit in billions of dollars (Chart F1: 2006–07; Chart F2: 2008–09). The A2/B2/C2 special-cause recipes are also given in the same prose.
- **Why** — Inherits the rationale of epic [#309](https://github.com/lddurbin/twelve_days_to_deming/issues/309) and parent [#313](https://github.com/lddurbin/twelve_days_to_deming/issues/313): SVG output is crisp at any zoom, adapts to dark mode via the CSS-filter mechanism from [#310](https://github.com/lddurbin/twelve_days_to_deming/issues/310), and turns chart edits into one-line code changes rather than new crop passes. The panel helper also unlocks Day 3.09's extended A3–F3 grid ([#345](https://github.com/lddurbin/twelve_days_to_deming/issues/345)) and individual-chart variants ([#346](https://github.com/lddurbin/twelve_days_to_deming/issues/346)) by reusing the same composition pattern. Two data-tier decisions are recorded below.
- **Data decision A — Simulated trio (A, B, C) uses fixed seeds** — Neave does not print the throw-by-throw / toss-by-toss / paddle-by-paddle values for the three "All-Knowing" processes. The R file simulates them under fixed seeds so the resulting chart shapes match Neave's published image: `set.seed(274)` for A (inherited from [#343](https://github.com/lddurbin/twelve_days_to_deming/issues/343)), `set.seed(151)` for B, `set.seed(88)` for C. Process B uses `rbinom(24, size = 25, prob = 0.5)` for B1 and a 14-toss steady-state followed by 10 tosses with coin counts 27, 29, ..., 45 for B2 — the special-cause recipe Neave describes. Process C uses `rbinom(24, size = 50, prob = 0.20)` for C1 (Neave's nominally 20%-red urn) and the same for the first 18 paddles of C2, with the final 6 paddles modelled as `2 * rbinom(6, size = 50, prob = 0.20)` to mimic "summing the two junior inspectors' counts." Seeds were chosen so each panel reproduces the visual pattern Neave's prose identifies — A2's step shift, B2's climb to the upper limit, C2's late spike. The same dataset feeds [#345](https://github.com/lddurbin/twelve_days_to_deming/issues/345) (the extended A3–F3 panel) so the page-19 ↔ page-23 internal consistency Neave's prose relies on holds across both pages.
- **Data decision B — Digitised trio (D, E, F) is visual digitisation, not historical truth** — The "serious" processes are real-world series whose raw values Neave does not reproduce: D is personal pulse data, E is a confidential Japanese case study, F is monthly BEA trade-deficit data Neave rounded for the chart. The 48 values for each of D, E, F in `R/data/day-03-six-processes.R` are **digitisations of the printed chart on page 19** — they reproduce the *visual shape* (D1 baseline + D2's beta-blocker drop in the final four days; E1's tight 15.91 baseline + E2's transient fault around points 10–12; F1's 2006–07 stability + F2's 2008–09 financial-crisis trough and partial recovery) without claiming to be the original BEA / case-study / personal values. The pedagogical signal — that all six stable charts look broadly alike and all six unstable charts differ wildly — is preserved either way.
